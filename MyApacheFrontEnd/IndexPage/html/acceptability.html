About Linguistic Acceptability:

It checks not only the grammar but also whether the sentence itself is "logically" correct.

For example, "John is reading a paper." is linguistically acceptable, but "A book is reading about John." is not linguistically acceptable, though there's no grammar mistake.


The input should be a single sentence.

Chrome is the recommended and only tested browser.

Text to summarize has to be encoded in UTF-8.
